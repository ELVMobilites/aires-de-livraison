📦 Points relais & Lockers — Import CSV
Les APIs des transporteurs (Mondial Relay, InPost, Chronopost…) ne sont pas accessibles publiquement.
Exportez la liste des points relais depuis le site de chaque transporteur, puis importez-la ici pour les visualiser sur la carte.
💡 Comment obtenir le CSV depuis chaque transporteur ? ▼
📦 InPost — inpost.fr → Export CSV dans l'outil pro / API partenaire
🟣 Mondial Relay — mondialrelay.fr → Espace pro → Télécharger la liste des Points Relais
🟡 Colissimo — laposte.fr → Open Data La Poste (data.gouv.fr)
🔵 Chronopost — chronopost.fr → Export depuis l'espace expéditeur pro
🔴 DHL — dhl.com → Localisateur de points → Export
🟠 Relais Colis — relaiscolis.com → Espace partenaire → Télécharger
Format CSV attendu : Nom, Adresse, Ville, Latitude, Longitude (séparateur , ou ;)
